## 🛠️ How to Use

1. **Clone the repo**
   ```bash
   git clone https://github.com/mohanadsabha/CS50x2022.git
   cd CS50x2022
   ```
2. **Navigate to a problem set folder**, e.g. `pset1/`, and run:
   - **C**: compile & execute
     ```bash
     make hello
     ./hello
     ```
   - **Python**: run script:
     ```bash
     python3 script.py
     ```
   - **Web (HTML/CSS/JS/Flask)**: open in browser or run Flask app.
